# Getting Started with Windows Performance Tweaks

Welcome to the Windows Performance & Optimization Tweaks repository! This guide will help you get up and running quickly.

## 🚀 Quick Start

### Prerequisites
- **Windows 10/11** operating system
- **Administrator privileges** to run the batch scripts
- A backup of your system (recommended) before applying changes

### Installation Steps

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/dev-fahim-code/windows-performance-tweaks.git
   ```
   Or download the ZIP file and extract it.

2. **Choose Your Optimization Script**
   - `network_optimize.bat` - For network and Wi-Fi performance
   - `max_perf_visuals.bat` - For UI responsiveness
   - `system_latency_perf.bat` - For CPU and system latency

3. **Run the Script**
   - Right-click the desired `.bat` file
   - Select "Run as administrator"
   - Wait for the script to complete
   - Restart your computer

## 📊 Performance Expectations

After running these optimizations, you should expect:

| Optimization | Expected Impact |
|---|---|
| Network Tweaks | 10-30% faster network speeds (depending on hardware) |
| Visual Tweaks | Noticeably faster UI responsiveness |
| Latency Tweaks | 5-15% reduction in input lag for games |

> **Note:** Actual performance gains depend on your hardware configuration, drivers, and current system state.

## 🔄 Reverting Changes

If you need to revert to default Windows settings:

1. Open **Settings** → **System** → **About** → **Advanced system settings**
2. Go to **Performance Options** and select "Let Windows choose"
3. Restart your computer
4. For network settings, you may need to manually reset them via Device Manager

Alternatively, restore from your system backup created before applying the tweaks.

## ⚠️ Important Warnings

- **Back up your system first** using Windows built-in backup or a third-party tool
- **Do not run multiple scripts simultaneously**
- **Administrator access is required**
- Test changes on a non-critical system first if possible
- Some changes may affect system stability if not properly understood
